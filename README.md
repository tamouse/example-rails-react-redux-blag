# example posts

- Rails 5.1 API
- React + Redux client

## Setting up:

Fork and clone repo:

    git clone git@github.com:YOURUSER/my-blag.git
    cd my-blag
    bin/setup

## Running in Development

Either:

    bin/rake start

or:

    bin/start

do the same thing. You can start it more directly with:

    bundle exec foreman start -p 3000
